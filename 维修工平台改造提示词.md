# 水电维修平台 - 维修工个人工作台改造提示词

## 项目背景

这是一个为维修工（父亲）量身定制的水电维修平台。父亲是一名水电维修工人，每天需要接听客户电话、安排上门维修、管理预约时间。平台帮助他：
- 让客户在线提交维修需求（代替电话接单）
- 查看维修内容、预约时间、客户电话和地址
- 管理每日工作日程，防止预约时间冲突
- 记录维修进度和备注

## 核心改造要点

### 1. 去除商业化元素
- **不展示任何价格/报价信息** — 维修费用由维修工与客户线下沟通
- 移除"价格透明"等营销话术
- 移除维修员排名、收入统计等管理功能
- 平台定位为"个人工具"而非"商业平台"

### 2. 单维修工模式
- 系统只有一个维修工账号（dad / 123456）
- 所有订单默认分配给该维修工
- 移除多维修工管理功能（添加/编辑/删除维修工）
- 管理后台简化为"维修工工作台"

### 3. 服务分类
```
水管：漏水、堵塞、安装、更换、其他
电路：开关不灵、插座故障、线路老化、灯具维修、其他
地暖：地暖安装、地暖维修、地暖清洗、管道改造、其他
其他：卫浴安装、龙头更换、安全检测、其他
```

### 4. 订单状态流程（简化）
```
待接单(pending) → 已接单(accepted) → 维修中(repairing) → 已完成(completed)
                                                                    ↗
                                                        已取消(cancelled)
```
移除了"上门中(going)"和"已评价(reviewed)"状态。

### 5. 时间段预约系统
- 客户选择上门日期 + 时段（如 09:00-11:00）
- 系统自动检查时间段冲突，防止同一时段接多个单
- 维修工可设置工作日（默认周一至周六）
- 维修工可设置休息日（特殊日期不接单）
- 维修工可修改维修时长，自动同步到日程

### 6. 紧急标记
- 维修工可标记订单为"紧急"
- 紧急订单在列表中置顶显示（红色标签）
- 工作台首页突出显示紧急订单数量

### 7. 客户信息一键触达
- 电话号码可点击直接拨打（`tel:` 链接）
- 地址可点击打开高德地图导航
- 客户昵称、手机号、地址一目了然

### 8. 维修进度记录
- 维修工可上传维修进度照片（最多5张）
- 时间线展示维修过程
- 维修备注可随时编辑保存

## 用户端功能

### 首页
- 服务项目展示（水管、电路、地暖、其他）
- 点击服务项目直接跳转报修页并预选分类
- 用户评价展示（从Mock数据动态加载）
- 消息通知铃铛（未读数角标）

### 报修提交
- 选择问题大类 → 小类
- 填写问题描述（最多500字）
- 上传现场照片（最多5张）
- 选择/填写服务地址（支持常用地址快捷选择）
- 选择上门日期 + 时段（自动检查可用性）
- 提交前再次检查时间段冲突

### 订单列表
- 按状态筛选（待接单/已接单/维修中/已完成/已取消）
- 搜索订单号/问题描述
- 显示预约时段和紧急标记

### 订单详情
- 状态时间线（4步：提交→接单→维修→完成）
- 订单信息（类型、描述、联系方式、时段）
- 服务地址
- 维修进度照片
- 服务评价（完成后可评价，支持标签选择）

### 个人中心
- 我的订单、地址管理、编辑资料
- 意见反馈、关于我们、服务条款
- 退出登录

## 维修工后台功能

### 工作台（首页）
- 统计卡片：待接单数、紧急订单数、今日订单数、已完成数
- 今日日程：按时间段展示当天订单，紧急标红
- 待接单列表：快速接单
- 问题类型分布图

### 订单管理
- 筛选：状态、类型、紧急
- 搜索：订单号、手机号、描述
- 操作：接单（选时段）、更新状态、查看详情
- 导出CSV

### 订单详情
- 状态操作：接单→开始维修→完成维修
- 紧急标记切换
- 修改维修时长/时段
- 电话一键拨打
- 地址一键导航
- 上传维修进度照片
- 编辑维修备注

### 日程管理
- 周视图日历，按时间段展示订单
- 点击日期查看详情
- 设置/取消休息日
- 右侧面板显示选中日期的订单列表

### 客户管理
- 客户列表（搜索、导出CSV）

### 工作设置
- 个人信息（姓名、电话、服务区域）
- 工作日设置（选择每周工作日、设置可用时段）
- 休息日管理（添加/移除休息日）
- 修改密码

## 技术栈

- Vue 3 + Vite
- Ant Design Vue 4
- Vue Router 4
- Mock数据（localStorage模拟后端）
